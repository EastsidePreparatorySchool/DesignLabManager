Laser Cutter.

The laser cutter uses a laser to cut materials or engrain an image or text into a material.

  • InkScape
01 Novice
-You are already a novice. Congrats!
02 Competent
-Know how to use both cardboard and wood.

 -Know the difference between vector and raster.

 -How to control the power and speed.

 -Flag problems like: Fire and smoke and strange head movements.
03 Proficient
-Know how to combine file designs.

 -Know how to diagnose communication errors.

 -Know how to use Acrylic (grades 9-12).
04 Advanced
-Know how to make multicolor cuts.

 -Be able to use other materials (leathers, plastic, metal).

 -Design and create a project that improves the MakerSpace.
05 Expert
-Know how to clean the hood and tray.

 -How to replace the lens.
