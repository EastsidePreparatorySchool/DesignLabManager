Levels.

All tools have the same ranking scheme, but different requirements for each tool. To view general guidelines click the rank.

Novice Competent Proficient Advanced Expert
Novice is the starting rank. The only thing that it requires is an intrest in the tool. Novices are generally not allowed to use tools, only spectate as someone with at least a Profision rank operates the tool. {{noviceLevels | safe}} Competent is the rank at which one understands the basics of operating the tool. Competent students are allowed to use the tool when supervised by an Expert. {{competentLevels | safe}} Proficient students must know how to operate the tool and must be able to do basic troubleshooting if something goes wrong. Proficient users are alowed to use the tool without supervision, however supervision is advised. {{proficientLevels | safe}} Advanced is a rank that is usually achieved by giving back to the tech community. This means that the user should be able to design and create a product that benifets the community. {{advancedLevels | safe}} Expert users must be able to maintainance the tool. These people are responsible for making sure that the tool stays in working condition. They are also responsible for certifying other users. {{expertLevels | safe}}
