Power Tools.

These sets of tools encompass a wide array of tools that require an aditional power source instead of manual power. These tools include: drills, cordless driver, heat gun, etc.

01 Novice
-You are already a novice. Congrats!
02 Competent
-Know the names of the tools.

 -How to load the bit into the tools.

 -How to control the speed and direction of the tools.

 -Know how to safely use the tools.

 -How to switch the battery packs and charge them if it's low.

 -Know what the tools are used for.
03 Proficient
-Tools permitted: corded drill and heat gun.

 -Know how to use the tools without help.

 -Supervise others must be in grade (9-12).
04 Advanced
-Know how to service the tools.

 -Design and create a project that improves the MakerSpace.
05 Expert
-There is no expert level for power tools.
